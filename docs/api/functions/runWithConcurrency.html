          items: T[] | Promise<T[]>,
          concurrency: number,
          processor: (item: T, index: number) => Promise<R>,
      ): Promise<R[]>

      Runs an asynchronous task on an array of items with a specified concurrency limit. Uses a slot-filling approach: as soon as one task completes, the next item starts processing, keeping all slots busy at all times.

      Type Parameters

      • T

        The type of items in the input array.

      • R

        The type of the result array items.

      Parameters

      • items: T[] | Promise<T[]>

        The array of items to process, or a promise that resolves to one.

      • concurrency: number

        The maximum number of concurrent tasks. If not finite or <= 0, processes all concurrently.

      • processor: (item: T, index: number) => Promise<R>

        The async function to execute for each item.

      Returns Promise<R[]>

      A promise that resolves to an array of results in the same order as the input items.
