    • auto<TTasks extends AutoTasks>(
          tasks: TTasks,
          options?: AutoOptions,
      ): CancellableHandle<AutoResult<TTasks>>

      Executes a dependency graph of async tasks.

      Each task can be:

      • A function (results, token) => Promise<value>
      • A tuple [dependencies, taskFn]

      Type Parameters

      • TTasks extends AutoTasks

      Parameters

      • tasks: TTasks

        Task map where keys are task names.

      • Optionaloptions: AutoOptions

        Configuration options, including cancellation and concurrency.

      Returns CancellableHandle<AutoResult<TTasks>>

      A cancellable handle resolving to task results by key.

      Example

      const handle = auto(
        {
          config: async () => ({ baseUrl: "/api" }),
          user: [["config"], async (results, token) => {
            await token.sleep(5);
            return { id: 1, url: `${(results.config as { baseUrl: string }).baseUrl}/users/1` };
          }],
          posts: [["user"], async (results) => {
            const user = results.user as { id: number };
            return [`post-of-${user.id}`];
          }],
        },
        { concurrency: 2 },
      );

      const result = await handle.promise;
      // { config: { baseUrl: "/api" }, user: { ... }, posts: ["post-of-1"] }
